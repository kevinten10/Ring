# DeepSeek

## 简介
DeepSeek是一个专注于代码和数学领域的大型语言模型系列，由DeepSeek AI开发。最新的DeepSeek V3采用了创新的混合专家系统(MoE)架构，拥有6710亿参数，在多个基准测试中展现出卓越的性能。

## 最新版本

### DeepSeek V3
- 6710亿总参数规模
- 370亿活跃参数/token
- 128K tokens上下文窗口
- MoE创新架构设计
- 多token预测能力
- 负载均衡优化

### DeepSeek R1
- 实时联网搜索能力
- 深度思考模式
- 代码生成和分析
- 数学问题求解
- 文档理解处理

### DeepSeek Coder
- 专注代码开发
- 多语言支持
- 代码补全
- 调试优化
- 文档生成
