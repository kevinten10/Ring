# 通义千问 (Qwen)

## 简介
通义千问(Qwen)是阿里云开发的大型语言模型系列，最新版本Qwen 2.5于2025年1月发布。该系列模型在多个中文基准测试中排名第一，全球排名第三，仅次于Anthropic和OpenAI的顶级模型。

## 最新版本

### Qwen 2.5
- 参数规模：0.5B-72B
- 训练数据：18万亿Token
- 支持29+种语言
- 上下文窗口：最高1M tokens
- 多模态理解能力
- 结构化数据处理

### Qwen Omni
- 多模态理解生成
- 支持文本/图像/语音/视频
- 4种自然对话音色
- 32K上下文窗口
- 批量处理能力
- 实时响应优化
