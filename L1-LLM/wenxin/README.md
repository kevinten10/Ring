# 文心一言 (ERNIE Bot)

## 简介
文心一言(ERNIE Bot)是百度公司开发的大型语言模型，基于ERNIE(Enhanced Representation through Knowledge Integration)架构。最新版本ERNIE 4.0于2024年9月发布，并将App版更名为"文小言"，定位为智能搜索助手。

## 最新版本

### ERNIE 4.0 Turbo
- 更快的响应速度
- 更强的性能表现
- 支持128K上下文窗口
- 实时联网搜索能力
- 深度思考模式
- 多模态理解能力

### ERNIE 4.0
- 8K/128K上下文窗口
- 多token预测
- 知识蒸馏技术
- 并行处理优化
- 动态路由机制
