# 代码解释和优化和扫描

------------------------

## 代码解释【支持良好】

对整个代码库的解释，可能差强人意。

但如果指定某个包或者类族，那么可以很好的给出解释。

并且可以在满足框架结构的情况下，给出实现类等。

------------------------

## 代码扫描【支持良好】

cursor具有推理能力，会自动分析代码依赖的包、继承关系。

从而更好的理解和扫描代码。

------------------------

## 代码优化【支持良好】

在给定具体代码上下文的情况下，可以完成良好的代码重构。

常见的重构思路为方法抽离，变量抽离等。

可以通过指定进一步的架构原则、设计模式来进行进一步的重构。
