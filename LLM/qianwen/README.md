# 通义千问

## 简介
通义千问是阿里云推出的大语言模型，专注于中文理解和商业应用。

## 特点
- 强大的中文理解能力
- 丰富的知识图谱
- 支持多模态交互
- 商业场景优化

## 版本
- 通义千问：基础版
- 通义千问Pro：专业版

## 使用方式
1. 访问：qianwen.aliyun.com
2. 使用阿里云账号登录
3. 选择合适的版本
4. 开始对话

## 适用场景
- 商业分析
- 文档处理
- 知识问答
- 行业应用
- 数据分析

## 优缺点
优点：
- 中文理解优秀
- 行业知识丰富
- 商业场景适配好

缺点：
- 创意类任务相对较弱
- 国际化程度有限
