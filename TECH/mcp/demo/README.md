# Weather MCP Demo

这是一个基于MCP框架的简单天气查询工具示例，展示了如何使用MCP协议构建AI工具与模型的交互。

## 目录结构
```
demo/
├── weather_mcp/
│   ├── __init__.py
│   ├── config.py        # 配置文件
│   ├── weather_api.py   # 天气API封装
│   └── mcp_script.py    # MCP脚本实现
├── rules/
│   └── weather.yaml     # MCP规则配置
├── main.py             # 主程序
└── README.md           # 说明文档
```

## 使用方法
1. 安装依赖
2. 配置API密钥
3. 运行示例程序
